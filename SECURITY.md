# Security Policy

## Reporting a Vulnerability

This library handles cryptographic operations and deletion attestation. Security is critical.

**Please do NOT open public GitHub issues for security vulnerabilities.**

**Email:** mansoor@ephemeralsocial.com

### What to expect
- Acknowledgment within 48 hours
- Assessment and severity determination within 1 week
- Fix timeline communicated based on severity

### What qualifies
- Cryptographic weaknesses (key management, encryption, signing)
- Deletion receipt forgery or bypass
- Key material exposure or recovery after destruction
- Timing attacks or oracle attacks
- Any issue that could undermine the deletion verification guarantee

## Disclosure Policy

We practice coordinated disclosure. We will credit reporters in security advisories unless they prefer to remain anonymous.
